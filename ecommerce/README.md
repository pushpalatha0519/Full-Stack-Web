# Full Stack Web Application (E-commerce)

## 🚀 Project Overview
This is a full stack web application built using React (frontend) and Spring Boot (backend). The application allows users to register, login, and view products similar to Amazon/Flipkart.

## 🛠️ Tech Stack
- Frontend: React
- Backend: Spring Boot
- Database: H2 (for development)
- Security: Spring Security + OAuth2
- Build Tool: Maven

## 🔐 Features
- User Registration & Login
- Role-Based Access Control (Admin/User)
- Product Management
- User Profile Management
- OAuth2 (Google Login - planned)

## 👤 Roles
- **Admin**: Add, update, delete products
- **User**: View products

## ⚙️ How to Run
1. Clone repository
2. Open in VS Code
3. Run `EcommerceApplication.java`
4. Open browser: http://localhost:8080
